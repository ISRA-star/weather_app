 🌤️ Weather App

A Flutter Weather App with Firebase Authentication built with Flutter & Dart.

 ✨ Features
- 🔐 Firebase Login & Signup
- 🌡️ Real-time Weather Data
- 🔍 City Search
- 📱 Beautiful UI Design

 🛠️ Technologies Used
- Flutter & Dart
- Firebase Authentication
- Weather API
- Android Studio
